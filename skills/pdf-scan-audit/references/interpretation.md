# 결과 해석 가이드

자동 검사가 잡는 "의심"은 다음 두 가지 케이스가 섞여 있다. 시각 검증으로 가른다.

## 회전 의심 페이지

**자동 판정 기준**: 텍스트 줄 bbox의 가로 합계 < 세로 합계 × 1.5

가능 원인:
| 원인 | 판정 방법 | 처리 |
|---|---|---|
| 책 디자인이 의도한 가로 일러스트 (지도·다이어그램) | 양옆 페이지 본문은 정방향이고, 의심 페이지가 오직 그림만 있고, 8개 일러스트 페이지가 모두 같은 방향 | OK. 정상 |
| 스캐너가 한 페이지만 잘못 회전 인식 | 양옆 페이지와 다르게 그 페이지만 회전. 헤더/푸터 텍스트가 회전된 상태 | 회전 후처리 필요 |
| 가로(landscape) 페이지를 portrait으로 강제 | 페이지 메타가 portrait인데 모든 텍스트가 90° 누워 있음 | 페이지 회전 후처리 |

**판정 절차**:
1. 의심 페이지 양옆 페이지(±1)를 같이 추출해 비교
2. 헤더/푸터 텍스트의 방향 확인 (정방향이면 그림만 회전된 디자인)
3. 같은 책 다른 의심 페이지와 일관성 확인 (모두 같은 방향이면 디자인 의도)

## 페이지번호 연속성 갭

**자동 판정 기준**: 헤더/푸터에서 추출한 숫자 시퀀스의 인접 차이 ≠ PDF 페이지 차이

가능 원인:
| 갭 크기 | 가능성이 큰 원인 | 처리 |
|---|---|---|
| 갭 > 50 또는 음수 | 본문 안 숫자(연도, 코드 줄번호, ISBN 일부 등)를 OCR이 페이지번호로 오인 | OCR 오인식. 진짜 누락 아님 |
| 갭 1~5 | 진짜 누락 가능. 또는 챕터 시작 페이지에 페이지번호 미인쇄 | 양옆 헤더 직접 확인 필수 |
| 갭 = 0 (같은 번호 반복) | OCR이 본문 줄번호를 잘못 잡음 | OCR 오인식 |

**판정 절차**:
1. 갭 양옆 PDF 페이지의 헤더 영역만 220 DPI로 재추출
2. 실제 인쇄된 페이지번호를 사람이 직접 읽음
3. 연속이면 OCR 오인식, 비연속이면 진짜 누락

## 페이지 크기 미세 변동

**자동 판정 기준**: 가장 흔한 크기에서 5% 이상 벗어난 페이지

| 변동폭 | 원인 | 처리 |
|---|---|---|
| 0.5~2% (수 mm 차이) | 스캐너 자동 크롭이 페이지마다 약간 다름 | 무시 가능. 가독성 영향 없음 |
| 5% 이상 한두 페이지 | 표지/속표지/접지 페이지 등 | 정상 |
| 5% 이상 다수 페이지 | 스캐너 설정 변경 또는 다른 책 합본 | 재스캔 검토 |

## 일관된 PDF↔책 페이지 오프셋

대부분 책은 PDF 1페이지가 표지, 그 다음 몇 페이지가 속표지·머리말·목차(로마자 번호 또는 번호 없음), 그 후 본문 1쪽이 시작된다.

**오프셋 자동 추정**: pagenum_check가 추출한 시퀀스에서 `PDF페이지 - 책페이지`의 mode를 잡으면 그게 본문 시작 오프셋.

오프셋이 처음부터 끝까지 같은 값으로 유지되면 → **누락 없음**의 강한 신호.
중간에 오프셋이 갑자기 바뀌면 → 그 지점에 누락 또는 중복 의심.
